How to use?

If you have a stream called mylecture which is being streamed, you can join the live stream by

  • using any player which can play RTMP streams, like VLC. Use this URL:
    rtmp://$DOMAINNAME/live/mylecture
    This method is the recommended as it has the lowest delay and is the most reliable of all.

  • using an MPEG-DASH capable player or browser. Use this URL:
    https://$DOMAINNAME/livedash/mylecture.mpd
    Some browsers (Firefox, Chrome) need a plugin to play this. It has higher delay (20sec).

  • using an HLS capable player or browser. Use this URL:
    https://$DOMAINNAME/livehls/mylecture.m3u8
    Some browsers (Firefox, Chrome) need a plugin to play this. It has higher delay (20sec). This format is your best chance for smartphones.

If you are past the live streaming event called mylecture, you can download MP4 videos from https://$DOMAINNAME/videos/mylecture

Have fun!
