Checklist before Publishing Streams

Before the lecture, do not forget to set

Publishing Token
Generate the publishing token and set it in OBS! You have to do this every day once.
Stream Key
Set the Stream Key for the live streaming event. DO NOT use the key of an other event! One key is needed for one chain of events as all individual events will be saved by a timecode under videos folder.
Tell your audience about the correct Stream Key! Everything for viewers on usage can be found at https://$DOMAINNAME.
Start Streaming
Videos are recorded by this server and can later be reviewed. However, you can make your own recordings for safety reasons. Keep in mind that live streams can be delayed by 20 seconds in a browser, 5 seconds in a dedicated player.
